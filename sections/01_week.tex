\subsection{Memory Layout}
Klassen werden auf dem Heap angelegt und sind implizit public. Structs werden auf dem Stack angelegt und sind implizit public. \\
\begin{minipage}{0,33\linewidth}
\begin{lstlisting}
// Example 1
public struct Student
{
	public Exam exam;
}
public class Exam
{
	public int grade;
}
\end{lstlisting}  
\end{minipage}
\begin{minipage}{0,33\linewidth}

\end{minipage}
